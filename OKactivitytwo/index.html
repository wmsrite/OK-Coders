A CSS Exercise

This activity is focusing on the use of CSS to style the webpage(s). With Cascading Style Sheets (CSS) the ability to add style and creativity to elements.


The purpose of CSS is to provide Web developers with a standard way to define, apply, and manage sets of style characteristics. CSS provides these capabilities through a technical model based on a hierarchical scope of effect, the separation of style from content, and a well-defined set of published standards. This section touches on each of these three topics in turn, focusing on how CSS can help the Web developer create maintainable, reusable code. Within this exercise we will be looking at the following subjects:

    Cascading
    Style
    Content
